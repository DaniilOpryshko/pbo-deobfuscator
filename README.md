# PBO File Reader & Rapifier

A Java-based tool for reading and unpacking PBO (Packed Binary Object) files, with support for rapified content
decompilation.

## Current Status

This project is currently under development and not feature complete. Future updates will include:

- Native compilation for improved performance
- Command line interface improvements
- Additional error handling and validation

## Usage

Currently, the application can be used to unpack PBO files and decompile rapified content:

1. Place your .pbo file in the application directory
2. Run the application
3. Unpacked contents will be saved in a directory matching the PBO filename

## Building

The project uses Java and can be built using Maven. A native compilation option will be added in future releases.

**Note:** This is a development version and may contain bugs or incomplete features.
